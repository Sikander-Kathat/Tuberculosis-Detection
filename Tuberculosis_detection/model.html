AI Models

DenseNet 121

DenseNet-121 can be utilized for tuberculosis (TB) detection from chest X-ray images by training it on a dataset containing labeled X-ray images. Due to its densely connected architecture, DenseNet-121 captures fine-grained features from the input images efficiently, making it suitable for medical image analysis tasks like tuberculosis detection.

Time taken: 10 minutes

Test accuracy: 93%

Lenet 5

LeNet-5 can be adapted for tuberculosis (TB) detection from chest X-ray images by training it on a dataset containing X-ray images labeled with TB-positive and TB-negative cases. The network learns to extract relevant features from the images during training and makes predictions based on these features during inference.

Time taken: 3 minutes

Test accuracy: 90.25%
